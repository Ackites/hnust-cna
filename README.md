## 功能

1. 自动登录校园网:
   + [x] `账号密码保存在本地，不会上传到服务器`
2. 自动断线重连
3. 自动检测更新
   + [x] `支持自动更新`
   + [x] `支持手动更新`
4. 自动检测网络状态
